# RANDOM LIBS in Java++

## 1. rand.integer(num. num);

```
@USING::STD
@USING::RAND

new.variables.in.stack random_int = rand.integer(1, 5);

print.consolewrite.puts(random_int); // 1-5

quit.program.uwu(0);
```

## 2. rand.floating(num, num);

```
@USING::STD
@USING::RAND

new.variables.in.stack random_float = rand.floating(2, 9);

print.consolewrite.puts(random_float); // 2-9

quit.program.uwu(0);
```

## 3. rand.ranges(start, stop, step);

``` 
@USING::STD
@USING::RAND

new.variables.in.stack random_ranges_code = rand.ranges(0, 21, 5);

print.consolewrite.puts(random_ranges_code); // 0-21 + step 5

quit.program.uwu(0);
```
